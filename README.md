顾名思义，心理学的本质是研究，分析，解决人的心理或心理问题。心理学首先面临的是两个问题，一是解决人的心理困惑，如失眠、焦虑、强迫、抑郁等等，二是开发人的潜能，让人的能力充分展示出来。

但在实际进程中，解决人的各种心理问题，已经步履维艰，迄今为止，虽然有了几大流派，400多重技术，但解决问题的效果并不是太好，多数的失眠，焦虑症，强迫症，抑郁症患者得不到有效地帮助，一生过着痛苦、无奈的生活。至于开发潜能，只有极少数，极少数的人能体验到，感受到，并能大幅度的提升学习成绩，提升工作业绩，让他们越来越自信，越来越优秀。

所以心理学要解决的问题很多，要通过进一步探讨，找出真正有用高效的解决心理困惑的方法，让更多的饱受痛苦的人尽快脱离痛苦，回归正常的生活，在此基础上，发挥潜能，让自己的生活更好，同时为家庭，为社会做贡献。
